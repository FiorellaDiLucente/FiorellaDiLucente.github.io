Santa Cruz river dams (Patagonia Argentina)

Info
Earth works - Early stage of construction, 2018.

Project duration: 72 months

BID Amount: $4.8 billion

The project consists in two hydropower dams along the Santa Cruz river and is the largest infrastructure project in Argentina nowadays. The install capacity will be 1,310 MW, with an average annual plant generation of 5,100 GWh.

Condo Cliff dam
Install capacity: 950 MW
Annual generation: 3270 GWh
Turbomachines: 5 Francis
Dam type: Concrete face rockfill dam (CFRD)
Dam height: 240 ft (73 meters)
Dam lenght: 6,562 ft (2,000 meters)

La Barrancosa dam
Install capacity: 360 MW
Annual generation: 1900 GWh
Turbomachines: 3 Kaplan
Dam type: Concrete face rockfill dam (CFRD)
Dam height: 135 ft (41 meters)
Dam lenght: 8,022 ft (2,445 meters)
